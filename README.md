# PRO MEDIA — بروميديا

Official website for PRO MEDIA and Executive Producer Yousef Shaheen.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
npm start
```

## Deploy

Deploy the `pro-media` folder to Vercel. No environment variables are required.
